MikoIDE is a simple IDE (Integrated Development Environment) built from scratch. It provides basic features for editing, linting, and running code.

Key features:
- File editing with syntax highlighting
- Linting for basic syntax errors
- Running code snippets
- Customizable layout
- Extensible with plugins

Roadmap:
- Add support for more languages
- Implement a more robust linting system
- Add features for debugging code
- Improve performance for large files
- Add support for version control

Technology Stacks:
React: WebUI Framework
CEF Chromium: Embedded WebView Browser C++
